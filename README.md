### 一键安装

```bash

git clone https://github.com/b6421582/NetHammer.git
cd NetHammer

# 2. 连接VPS并上传
ssh root@your-vps-ip
scp -r NetHammer/* root@your-vps:/opt/nethammer/

cd /opt/nethammer
bash install_nethammer.sh

```

### 命令参数

```bash
python3 quick_attack.py <目标IP> [选项]

  -p, --port      目标端口 (默认: 80)
  -t, --time      攻击时间/秒 (默认: 300)
```
